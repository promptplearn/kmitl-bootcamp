Build a single, self-contained HTML file for a clickable mockup of: [your idea]. Requirements: - Everything inline — CSS in a <style> tag, JS in a <script> tag. - NO external libraries, CDNs, web fonts, or image URLs. Use emoji or CSS shapes instead. - It must work offline as one saved .html file. - Modern, clean, mobile-friendly. Make the key buttons actually do something on click. Output ONLY the HTML code, nothing else.
